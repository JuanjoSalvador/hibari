### Requirements

- [git] `>2.0.0`
- [node] `>7.0.0`
- [yarn] `>1.0.0`

### Setup

1. `git clone -b new git@github.com:wopian/hibari.git`
2. `git checkout -b myFeature`
3. `yarn install` Install dependencies
4. `yarn dev` Build & run a local dev server
